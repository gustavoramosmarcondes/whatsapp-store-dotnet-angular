# Visão do Sistema

## Contexto
Este sistema foi criado para atender pequenos negócios que vendem produtos físicos e realizam pedidos via WhatsApp.

## Problema
Muitos pequenos empreendedores não possuem um site próprio e organizam pedidos de forma manual pelo WhatsApp.

## Solução
Um site público com catálogo de produtos e carrinho de compras, onde o pedido é enviado automaticamente para o WhatsApp da empresa.

## Público-alvo
- Pequenos comércios
- Negócios familiares
- Lojas locais

## Escopo inicial
- Cadastro de produtos
- Cadastro de categorias
- Listagem pública de produtos
- Carrinho de compras
- Integração com WhatsApp

## Fora do escopo (por enquanto)
- Pagamento online
- Login de cliente
- Entrega automatizada
